使用hexo和GitHub搭建个人博客

Posted on 5月 25 2022

关于第一次写博客的一些想法

&ensp;早在大学的时候就一直想要自己来搭建一个网站玩一玩，个人博客就是最好的选择，但是为什么到现在为止我才开始着手于这件事情呢，因为大学的期间个人原因和家庭原因导致我对于学业的疏忽，并未将全部的精力投身于学业，导致自己能力欠缺，回想起来可真是遗憾，但是好后来我遇见了很多优秀又愿意帮助我的人，让我渐渐回到我最初热爱的行业，在我有充足的时间后，我又拾起了最开始的念头—搞一个个人博客记录我的工作，于是你就看见了现在的这段文字：）

作为第一篇博客就是介绍这个网站是如何搭建的吧。

技术介绍

该网站的搭建依靠的工具有：1.node.js;2.Git;3.hexo
Hexo是一个高速，简介且高效的博客框架，它可以方便的生成静态网页托管在GitHub和Coding上。点击 hexo官网，你可以看到更多介绍和信息 。
Hexo 只需要在本地安装即可，不需要在服务器上面安装。我们只需在本地编写 Markdown 文章。通过 hexo 命令即可让 Hexo 会帮我们生成静态的 Html 页面，通过将该 Html 文件上传到我们的服务器，这里我使用的服务器用的是 GitHub。
所以在Github上面并没有我们的具体的markdown的文档，因为我们都是生成的html后再上传的。
